# So-dart-sdk

A full sdk dart for Saas-Office api generated by OpenApi Generator.

## Generate the SDK

**Requirements:**
- `brew install php` installed
- `brew install composer` installed
- `brew install openapi-generator` installed
- dart cli

**Generate the SDK:**
```bash
./generate.sh
```

**Run g.dart files:**
```bash
dart run build_runner build
```
